Neural Network Wizard

The Neural Network Wizard is displayed during the application startup and when the New menu action is used.

The left column displays a list of all supported neural network types. By selecting one of the types, a list of examples is loaded and shown in the right column. To load one of the example networks, either double click it, or select it and press the Finish button. To create a custom neural network of the selected type, check the Create a new network checkbox and press the Next button.

The Load from File button displays the same dialog as if the Open menu action was used. It is provided here for convenience.
