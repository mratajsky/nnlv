ADALINE

The ADALINE wizard provides the following customization options:

  • Name of the new network: choose a name for the newly created network
  • Neurons in the input layer: number of input neurons
  • Neurons in the output layer: number of output neurons
  • Initial connection weights: choose whether to have the initial connection weights initialized to random values in the given range, or to use the selected value to initialize them

The Training Options section additionally contains the following options:

  • Learning rate: the learning rate constant in the range (0, 1]
  • Training sample selection order: choose whether to select the training samples in a random order during training, or select them in the order they appear in the training table
