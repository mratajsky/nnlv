Window Menu

The Window menu category contains the following options:

  • Full Screen: enable or disable full screen mode
  • Options: open dialog window to adjust the program options (see below);

Options

The following program options are provided:

  • Ask to save modified neural networks when closing windows: when enabled, the program asks to save unsaved neural networks when a window with unsaved changes is closed
  • Mark networks as modified when only connection weights have been changed: when enabled, the neural network will be considered modified when connection weights change; enabling this option makes the program ask to save unsaved changes when a network is changed only by training or by user selecting different weights using the neuron options dialog
