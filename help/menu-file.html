File Menu

The File menu category contains the following options:

  • New: open the network wizard to create a new network
  • Open: load a network saved in an XML file
  • Save: save the current network to an XML file; this action is disabled if the current network was loaded from a file, but currently has no unsaved changes
  • Save As: save the current network to the chosen location; this action is always enabled
  • Close: close the current window
  • Quit: close all windows and quit the application

Each program window may or may not be associated with a file. If it is, the name of the file is written in the window title bar and the Save action saves changes to this file.
