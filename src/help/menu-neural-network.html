Neural Network Menu

The Neural Network menu category is only enabled if there is a neural network loaded in the program window. It contains the following options:

  • Configure Training Set: open the training set configuration dialog
  • Compute Custom Input: open a dialog to enter custom input; the given input is then computed by the current neural network
  • Rename Network: open a dialog to rename the network
  • Reset Connection Weights: open a dialog to reset the connection weights
  • Visualize Connection Weights: choose whether to adjust the connection line thickness to display the weight proportions
  • Visualize Input Values: choose whether to visualize input values using text and background color; this option affects all input neurons
  • Visualize Neuron Values: choose whether to visualize neuron values using text and background color; this option affects all non-input neurons
  • Reset Network View: reset the network diagram and all visualization panels to their default state; this does not change the neural network, it only removes information about the last processed training sample after
